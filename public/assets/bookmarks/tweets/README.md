Local tweet media lives in this folder.

Recommended conventions:
- Keep assets small and cache-friendly.
- Prefer `webp` for posters/images.
- Prefer `mp4` or `webm` for playable tweet previews.
- Keep filenames stable so the curated dataset can reference them directly.

Current media-ready card:
- `performance-video-preview.mp4`
- `performance-video-preview-poster.svg`

If the video file is missing, the UI falls back to a lightweight placeholder card instead of loading any third-party embed.
